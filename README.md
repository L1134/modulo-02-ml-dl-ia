## 📌 Sobre este módulo

Fundamentos de ML, Deep Learning e IA com foco em aplicação web usando **TensorFlow.js**. Aprendemos como redes neurais funcionam de verdade — desde tensores até treinamento e predição.

---

## 🧠 O que aprendi

- ✅ Diferença entre **IA, Machine Learning e Deep Learning**
- ✅ Como redes neurais funcionam: entrada → camadas ocultas → saída
- ✅ **Tensores**: representação numérica de dados
- ✅ **Normalização**: mapear valores para o intervalo [0, 1]
- ✅ **One-Hot Encoding**: transformar categorias em vetores binários
- ✅ **Transfer Learning**: reaproveitar modelos já treinados
- ✅ **Teachable Machine**: treinar modelos no-code com webcam
- ✅ Funções de ativação: **ReLU** (camadas ocultas) e **Softmax** (saída)
- ✅ Otimizador **Adam** e loss **Categorical Cross Entropy**
- ✅ Criar, treinar e testar uma rede neural com **TensorFlow.js**
